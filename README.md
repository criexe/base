## File Permissions
| File & Folder | Permission |
| ---           | ---        |
| *             | 777        |
| /sysdata/*    | 777        |
| /contents/*   | 777        |

### Set Permissions
`cd /path/to/project`
`chmod -R 755 *; chmod -R 777 sysdata; chmod -R 777 contents`
